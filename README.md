# webStudy3-toyProject
토이프로젝트 '**설문폼폼폼**'

✔설문조사 품앗이 사이트. 다른 사람의 설문조사를 참여함으로써 포인트를 쌓을 수 있고, 이 포인트로 자신의 설문조사도 올릴 수 있는 사이트. 

## 기술스택
**Client**
- React

**Server**
- Spring Boot
- MySQL
